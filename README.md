# SLA Mothership Companion

Draft companion module for `sla-mothership`.

When enabled in an `sla-mothership` world, the module seeds:

- SLA skills
- SLA species notes
- SLA training packages
- SLA weapons
- SLA armour
- SLA general gear
- SLA Ebb abilities
- SLA utility macros
- A starter operative squad for first-pass testing
- Tier data for Ebb powers so SLA Mothership can prompt for intensity on use

Artwork paths currently reuse the installed `sla-industries-compendium` module assets.
